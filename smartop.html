website
Product Design - UX - UI - Product Management

I came in part way through this project at Opolis, and had to catch up quickly. An opportunities connection protocol for job seekers, recruiters, and employers, SmartOp seeks to incentivice all parties through tokens and game theory, verified by the Ethereum blockchain, so that all parties benefit from an otherwise (generally) misaligned recruitment market.

The project used identity and wallet web3 tech projects uPort and Metamask, which required detailed and unique UX. For a number of reasons we have shelved this project for now with hopes to revive it in the future.
